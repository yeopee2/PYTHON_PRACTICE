# PYTHON_PRACTICE

## Summary

This repository contains my Python study materials and assignments.  
I practiced:

- basic Python syntax and problem solving
- string/list processing and function design
- simple object-oriented programming
- exploratory data analysis with Pandas
- machine learning and neural network basics
- text preprocessing with NLTK
- network analysis with NetworkX
- web crawling with Selenium and BeautifulSoup

## Details

# python_basic.html
This file covers introductory Python exercises on basic syntax, simple functions, and string/list handling. Through these exercises, I learned how to write small utility functions, apply conditional logic, and get comfortable with core Python expressions and data manipulation.
# python_basic_2.html
This file contains another set of beginner-level Python exercises focused on applying core syntax to simple problem-solving tasks. It helped me reinforce fundamental concepts such as indexing, slicing, type conversion, and step-by-step algorithmic thinking.
# python_math.html
This file includes small class-based exercises, such as a circle class for computing circumference and area and a times_table class for printing multiplication tables. I used it to learn how Python classes, methods, and reusable logic are structured.
# python_EDA.html
This file focuses on exploratory data analysis using a medical dataset loaded with Pandas. I examined the dataset structure, checked missing values, compared hospital categories, and created boxplots, which helped me practice basic data cleaning, summarization, and visualization workflows in Python.
# python_ML(keras).html
This file is an introductory machine learning practice notebook centered on Keras. It helped me become familiar with how a basic machine learning workflow is organized in Python, including model-oriented coding structure and beginner-level deep learning practice.
# python_txet_preprocessing.html
This file focuses on text preprocessing with Python and NLTK. I practiced cleaning raw review text, creating cleaned text columns, and applying common NLP steps such as stemming, lemmatization, and part-of-speech tagging, which helped me understand the standard preprocessing pipeline for text data.
# python_Network_analysis.html
This file covers basic network analysis using product co-occurrence data. I learned how to represent relationships as a graph and analyze node importance using measures such as betweenness centrality, closeness centrality, and degree centrality.
# python_text_crawling.html
This file focuses on web crawling and data collection. I gathered presentation titles, cleaned the collected text, and organized the results into Pandas DataFrames by semester, which helped me learn the basics of scraping, text extraction, and structured result storage.
